🏨 Hotel Booking Analysis


📌 Descripción del proyecto

Este proyecto analiza un conjunto de datos de reservas hoteleras con el objetivo de identificar patrones y diferencias entre dos tipos de hoteles: Resort Hotel y City Hotel.
El análisis se enfoca principalmente en tasas de cancelación, tiempo de anticipación de reserva (Lead Time) y precio promedio por noche (ADR), utilizando Python y librerías de análisis de datos.

🎯 Objetivos

Comparar las tasas de cancelación entre hoteles Resort y City.

Analizar cómo el tiempo de anticipación influye en la cancelación de reservas.

Evaluar las diferencias en el precio promedio por noche (ADR) entre ambos tipos de hotel.

Visualizar los resultados mediante gráficas para una mejor interpretación.

🛠️ Herramientas utilizadas

Python

Pandas – manipulación y análisis de datos

Matplotlib – visualización de datos

Jupyter Notebook

📊 Análisis realizado

El proyecto responde a las siguientes preguntas clave:

¿Qué meses concentran mayor número de reservas?

¿Existen diferencias en las reservas entre Resort y City Hotel?

¿Qué tipo de hotel presenta una mayor tasa de cancelación?

¿El tiempo de anticipación de la reserva influye de manera distinta en las cancelaciones?

¿Qué diferencias existen en el precio promedio por noche (ADR) entre ambos hoteles?

Para el análisis del ADR, se excluyeron las observaciones con ADR = 0, ya que estas corresponden a reservas sin ingreso real y podrían distorsionar el valor promedio. Las reservas canceladas con cobro fueron mantenidas en el análisis.

📈 Principales conclusiones

El City Hotel presenta una tasa de cancelación mayor en comparación con el Resort Hotel.

Las reservas realizadas con mayor anticipación tienden a mostrar tasas de cancelación más altas, aunque no se puede afirmar una relación causal directa.

A pesar de la intuición inicial, el City Hotel presenta un ADR promedio mayor, lo cual puede explicarse por una mayor estabilidad en las tarifas urbanas y la estacionalidad del Resort.
